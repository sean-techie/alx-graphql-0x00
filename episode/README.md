# Episode Queries - ALX GraphQL Project

This directory contains GraphQL queries and output files related to **episodes** and **characters** for the ALX GraphQL exercises.

## Files

### GraphQL Query Files
- `episode-page-1.graphql`  
  Query to fetch a specific episode by its ID. Retrieves the following fields:
  - `id`
  - `name`
  - `air_date`
  - `episode`

- `characters-page-2.graphql`  
  Query to fetch the list of characters on **page 2**. Retrieves:
  - `id`
  - `name`
  - `status`
  - `image`

- `characters-page-3.graphql`  
  Query to fetch the list of characters on **page 3**.

- `characters-page-4.graphql`  
  Query to fetch the list of characters on **page 4**.

### Output JSON Files
- `characters-page-1-output.json`  
- `characters-page-2-output.json`  
- `characters-page-3-output.json`  
- `characters-page-4-output.json`
